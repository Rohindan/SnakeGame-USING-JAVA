# 🐍 Snake Game in Java

A classic **Snake Game built using Java Swing**.

I originally created this Snake Game in Python using Tkinter. As a learning challenge, I rebuilt the same game in Java by understanding the Python logic and translating it into Java concepts.

This project helped me practice Java Swing, object-oriented programming, keyboard events, game loops, collision detection, and working with `ArrayList`.

## 🎮 Features

* Snake movement using arrow keys
* Random food generation
* Snake grows after eating food
* Score tracking
* Wall collision detection
* Self-collision detection
* Game Over screen

## 🛠️ Technologies Used

* Java
* Java Swing
* AWT
* `ArrayList`
* `Random`
* Swing `Timer`

## 🎯 Controls

| Key            | Action     |
| -------------- | ---------- |
| ⬆️ Up Arrow    | Move Up    |
| ⬇️ Down Arrow  | Move Down  |
| ⬅️ Left Arrow  | Move Left  |
| ➡️ Right Arrow | Move Right |

## 📚 What I Learned

While building this project, I learned and practiced:

* Creating classes and objects in Java
* Using constructors and the `this` keyword
* Creating custom `JPanel` components
* Drawing shapes using `Graphics`
* Handling keyboard input using `KeyAdapter` and `KeyEvent`
* Creating a game loop using Swing `Timer`
* Using `ArrayList` to manage the snake's body
* Detecting collisions with walls, food, and the snake's own body
* Translating programming logic from Python to Java

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project in your preferred Java IDE.
3. Make sure Java is installed on your system.
4. Run `App.java`.
5. Use the arrow keys to control the snake.

## 💡 About This Project

This project was a personal learning challenge. Instead of following a Java Snake Game tutorial or copying a complete Java solution, I used my previously created Python Snake Game as the main reference and rebuilt the logic in Java.

During the development process, I used **ChatGPT as a learning assistant** whenever I needed help understanding the Java equivalent of Python concepts, such as `global` variables, lists and `ArrayList`, random number generation, keyboard events, drawing with `Graphics`, and using Swing `Timer`.

I also used ChatGPT to review my code, identify mistakes, and explain why certain changes were needed. The actual project was built step by step by translating the logic I already understood from Python into Java.

The goal was to strengthen my understanding of Java, improve my problem-solving skills, and learn how to transfer programming concepts from one language to another.
